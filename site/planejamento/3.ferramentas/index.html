    Ferramentas

    1. Introdução

       As ferramentas serão softwares que serão utilizados pela equipe com os objetivos de melhorar o desenvolvimento, comunicação e o gerenciamento do projeto.

    2. Ferramentas utilizadas

       Todas as ferramentas utilizadas pelo grupo estão descritas logo abaixo na tabela 1.

    Tabela 1: Ferramentas utilizadas no projeto

    Logo Ferramenta Finalidade
    Github Será utilizado como repositório dos documentos produzidos e para o versionamento dos mesmos.
    Telegram Utilizado para comunicação entre o grupo e com o monitor
    Lucidchart Será utilizado para a criação dos diversos diagramas e esquemas necessários para a modelagem dos Requisitos
    Youtube Utilizado para armazenar as gravações das apresentações dos pontos de controle
    Microsoft Teams Utilizado para as apresentações e reuniões do time.
    Visual Code Studio Utilizado para elaboração dos documentos.
    Figma Utilizado para elaborar os protótipos de média e alta fidelidade
    This Person Does Not Exist Inteligência artificial utilizada para gerar imagens de pessoas aleatórias para a criação de personas
    Zenhub Utilizado para gerenciar o fluxo de trabalho
